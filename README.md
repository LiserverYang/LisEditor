# LisEditor

[简体中文](./README_ZHCN.md)

This is a easy open-source text editor repository.

<hr>

## Build

- Windows:
    ```shell
    .\\build.cmd
    ```

- MacOS or Linux:
    It can't build on MacOS or Linux. But you can also use wine.

<br>

## How to use

<br>

**./liseditor** **./a.cpp[FileName]** **-ro[readonly]**

- Press key **CTRL + S** to save change.
